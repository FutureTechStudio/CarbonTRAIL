# UI, UX and Responsive Requirements

## Visual direction
Keep the current CarbonTrail AI nature-based theme. It should feel premium, calm, hopeful, and intelligent.

## Metaphors
- Daily footprint = winding trail
- Weekly = forest of 7 trees
- Monthly = valley
- High impact = smoke/haze/dry soil
- Positive progress = trees/flowers/clear sky
- Transport = road smoke
- Food = plate/farm patch
- Energy = house haze
- Delivery = parcel trail
- Waste = soil/recovery patch
- Estimated data = faded/dotted
- Confirmed data = solid

## Mobile navigation
Bottom nav:
- Today
- Check-in
- Forest
- Simulate
- Profile

## Desktop navigation
Sidebar:
- Today
- Check-in
- Weekly Forest
- Monthly Valley
- Simulator
- Profile

## Welcome
Keep current stats as-is:
- 2.4M Trails tracked
- 84% Found new insights
- 12 kg Avg. weekly saved

Include:
- Start as Guest
- Sign in
- local guest data note

## Today Dashboard
Mobile order:
1. metrics
2. trail
3. Sprout insight
4. LeafPoints/Green Impact
5. activity list

Desktop:
- left: large trail + activity timeline
- right: Sprout insight, LeafPoints, Green Impact, confidence/profile card

## Check-in
Mobile:
- chat messages
- chips
- sticky input
- parsed result below

Desktop:
- left: chat
- right: parsed preview and profile context

## Profile
Rename/position as My Carbon Memory.

Sections:
- Core Profile
- Work & Commute
- Food Routine
- Energy Routine
- Shopping & Waste
- Learned Patterns
- Default Daily Emissions
- Data Confidence

## Weekly Forest
No graphs. Use trees.

Tree states:
- green = low impact/confirmed
- medium = moderate
- smoky/dry = high impact
- faded = estimated

## Accessibility
- chips are buttons
- visible focus ring
- aria labels for icons
- status text not color-only
- semantic headings
- labelled inputs
- good contrast
- reduced motion support

## Motion
Use subtle:
- smoke drift
- leaf collection
- tree growth
- node glow

Avoid:
- video
- large Lottie
- heavy parallax
- distracting animation

## Copy tone
Friendly, practical, non-judgmental.

Good:
> Today was a smoky trail, but you shared your day clearly. That helps Sprout improve your Carbon Memory.

Bad:
> You failed to be eco-friendly today.
